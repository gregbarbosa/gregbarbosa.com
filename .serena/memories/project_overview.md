# Project Overview

**Project Name:** gregbarbosa.com  
**Purpose:** Astro-based personal blog theme (Saral Theme) focusing on simplicity and accessibility  
**Tech Stack:**

- Astro v5.16.6
- TypeScript 5.3.2
- Tailwind CSS v4 (via @tailwindcss/vite)
- Starwind UI component library
- astro-icon for Iconify icons
- MDX for rich content

**Key Integrations:**

- @astrojs/sitemap, @astrojs/rss
- astro-embed, lite-youtube-embed
- Custom remark/rehype plugins for reading time and modified dates
