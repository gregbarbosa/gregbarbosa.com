# Suggested Commands

## Development

- `npm run dev` or `npm run start` - Start development server
- `npm run build` - Type check and build for production
- `npm run preview` - Preview production build

## Code Quality (Pre-commit Workflow)

1. `npm run format:check` - Check formatting
2. `npm run lint` - Lint code
3. `npm run astro check` - Type checking
4. `npm run build` - Build project

## Quick Fixes

- `npm run format` - Auto-format code
- `npm run lint:fix` - Auto-fix linting issues

## System Commands (macOS/Darwin)

- Standard Unix commands: `git`, `ls`, `cd`, `grep`, `find`
- Package manager: npm (defined in package.json)
- GitHub CLI: `gh` for PR/issue management
